# shell scripts

to run this shell files
1. open your favourite terminal
2. clone this repo
    ```
    git clone https://github.com/ashokide/scripts.git
    ```
3. move into scripts folder
    ```
    cd scripts
    ```
4. make the files executable
    ```
    chmod +x *
    ```
5. execute the file
    ```
    ./<file-name>.sh
    ```
    example:
    ```
    ./scripts.sh
    ```
